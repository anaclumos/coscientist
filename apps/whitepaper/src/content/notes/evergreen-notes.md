# Evergreen Notes

Evergreen notes are written and organized to evolve, contribute, and accumulate over time, across projects. This is in contrast to notes that are tied to a specific project or time period.

## Key Principles

1. **Atomic** - Each note should capture exactly one idea
2. **Concept-oriented** - Notes should be about concepts, not sources
3. **Densely linked** - Notes should be connected to related ideas
4. **Written for yourself** - Don't worry about explaining everything
